## 📌 Project Overview

Royal Taste Restaurant Website is a professional business website designed for a restaurant to showcase its menu, attract customers, increase reservations, and improve online presence.

The website features a modern UI, responsive design, menu filtering, food recommendations, reservation management, customer reviews, and more.

## 🚀 Features

### 🎨 User Interface

- Modern Responsive Design
- Mobile Friendly Layout
- Glassmorphism Effects
- Smooth Scrolling
- Attractive Hero Section
- Loading Screen Animation
- Dark Mode / Light Mode

### 🍕 Restaurant Features

- Interactive Menu Section
- Menu Search Functionality
- Category Filtering

- Online Table Reservation Form

- Newsletter Subscription

### 💾 Local Storage Features

- Stores Reservations
